# Monitorcillo de la Economía Argentina - Documentación

Bienvenido a la documentación oficial del **Monitorcillo**. 
Este proyecto fue diseñado bajo una premisa estricta de minimalismo: "En ingeniería la mejor pieza es la que no existe".

## Archivos de Documentación

- [Arquitectura (architecture.md)](./architecture.md): Detalles sobre la filosofía cero-database y elección del stack.
- [Interacciones y Navegación (features.md)](./features.md): Detalle del flujo de navegación y click en la tabla.
- [API Reference (api.md)](./api.md): Documentación de los endpoints internos para automatización.
- [Despliegue y Uso (deployment.md)](./deployment.md): Cómo correr el proyecto localmente y en producción.

## Estilo Visual

El diseño sigue una estética **Imperial**, priorizando colores de alto contraste pero elegantes:
- **Azul Imperial:** `#00143F` (Fondo primario)
- **Dorado Imperial:** `#FFD700` (Textos primarios, acentos)
- **Celeste/Cyan Imperial:** `#00BFFF` (Textos secundarios, líneas de gráficos secundarias)

Todo el estilo está centralizado usando Native CSS + Tailwind CSS v4 para mantener la huella del cliente lo más pequeña posible.
